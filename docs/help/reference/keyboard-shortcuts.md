# Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd>+<kbd>N</kbd> | Clear the brief. |
| <kbd>⌘</kbd>+<kbd>↩</kbd> | Generate. |
| <kbd>Esc</kbd> | Cancel generation. |
| <kbd>⌘</kbd>+<kbd>S</kbd> | Save selected card to `history/`. |
| <kbd>⌘</kbd>+<kbd>E</kbd> | Export selected card to a file. |
| <kbd>⌘</kbd>+<kbd>I</kbd> | Open the CSV ingest dialog. |
| <kbd>⌘</kbd>+<kbd>R</kbd> | Open the Refine from feedback dialog. |
| <kbd>⌘</kbd>+<kbd>L</kbd> | Show last LLM prompts. |
| <kbd>⌘</kbd>+<kbd>,</kbd> | Open Preferences. |
| <kbd>⌘</kbd>+<kbd>Q</kbd> | Quit. |

## Inside the Brand Manager

| Shortcut | Action |
|----------|--------|
| <kbd>⌘</kbd>+<kbd>S</kbd> | Save edits to the current profile. |
